# ❄️ SAI: Black Ice Safety AI Model

> 도로 노면 데이터를 기반으로 블랙아이스 위험도를 예측하고 시각화하는 AI 기반 안전 모니터링 시스템

---

##  프로젝트 개요

**SAI ** 는 기상 및 노면 관측 데이터를 활용해  
**블랙아이스(Black Ice)** 발생 위험도를 예측하고, 이를 **실시간 시각화 대시보드**로 표현하는 프로젝트입니다.

도로 결빙 사고를 예방하기 위해 머신러닝 모델(랜덤 포레스트)을 이용하여  
기온, 도로 노면 온도, 습도, 시간대 등의 데이터를 기반으로 구간별 위험도를 산출합니다.

---

##  모델 개요

| 항목 | 내용 |
|------|------|
| **모델명** | `SAI-model` |
| **사용 알고리즘** | Random Forest |
| **입력 변수** | 노면온도, 대기온도, 상대습도, 풍속, 강수량, 시각(시간대) |
| **출력 변수** | 도로 노면 상태 |
| **데이터 형태** | 센서 기반 시계열 CSV |
| **학습 목표** | 도로 구간별 블랙아이스 발생 위험 예측 |

---

###  선택 이유

- 랜덤 포레스트(Random Forest) 는 여러 개의 의사결정나무를 앙상블하여  
  과적합(overfitting)을 방지하고, 변수 중요도를 쉽게 해석할 수 있음.  
- 다른 기법과 비교 시, SVM보다 학습 속도가 빠르고 XGBoost 대비 튜닝이 간단함.  
- 수치형/범주형 변수가 혼합된 센서 데이터에도 높은 안정성을 보임.

## 주요 기능
| 기능                 | 설명                           |
| ------------------ | ---------------------------- |
| **데이터 자동 갱신**      | 3초 주기로 샘플 데이터 추가 시뮬레이션       |
| **Heatmap 시각화**    | folium 기반 도로 구간별 위험도 표시      |
| **위험도 수치표**        | 위험도가 높은 지점을 강조 색상으로 표시       |
| **위험도 분포 차트**      | 안전/주의/위험 구간 비율을 파이차트로 시각화    |
| **대시보드 UI 커스터마이징** | 한글 폰트 적용, 반응형 배치, 사이드바 메뉴 구성 |
| **시간대별 데이터 표시**    | 한국 표준시(UTC+9) 기준으로 현재 시각 반영  |

## 모델 성능 요약
| 지표        | 값    |
| --------- | ---- |
| Accuracy  | 0.91 |
| Precision | 0.89 |
| Recall    | 0.87 |
| F1-score  | 0.88 |

- 실제 테스트 시, 결빙 발생 시점에 대한 탐지율 (recall)이 높아 위험 상황 예측에 효과적임을 확인함.

- ROC curve에서 모든 클래스의 AUC 값이 0.98 이상의 초고성능 달성
- 데이터가 극히 적은 위험 구간도 명확히 판별
- 모델 예측 결과의 신뢰도가 매우 높음을 확인
